Vishwa 301 Redirects provides an easy method of redirecting requests to another page on your site or elsewhere on the web with 301 redirect.

== Description ==
Add pages to rediect wit header code 301.
It is useful when you have shifted to new hosting and is not able to preserve old URL structure.
It helps to you and to your visitor to redirect to those pages where you want.

== Installation ==

1. Upload Vishwa 301 Redirects to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add redirects on the Settings > Vishwa 301 Redirects page.
